## Como cadastrar um usuário

1. No menu lateral, vá até a aba **Usuários**

   <img
     height="200"
     style={{ borderRadius:"0.5rem" }}
     src="/images/configuracoes/usuarios/user-menu.png"
   />
2. Clique no botão **Adicionar usuário**

   <img
     height="200"
     style={{ borderRadius:"0.5rem" }}
     src="/images/configuracoes/usuarios/add-user.png"
   />

<Warning>
Para pode adicionar um novo usuário você dever do tipo usuário **administrador**  
</Warning>

3. Preencha as informações necessárias no formulário.

- Nome, email, função: são itens obrigatórios.
   <img
     height="200"
     style={{ borderRadius:"0.5rem" }}
     src="/images/configuracoes/usuarios/user-form.png"
   />

<Info>
  As funções são vendedor e administrador. Sendo o administrador responsável por acessar as páginas de **[Integração](/configuracoes/integracoes)** e usuários
</Info>
